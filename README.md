# scorer.rb

Ruby implementation of scorer (https://github.com/alfie-max/scorer) to fetch live scores and send notification

## How to run the program

```sh
$ git clone https://github.com/coderhs/scorer-rb.git
$ cd scorrer-rb
$ bundle
$ ./scorer
```
